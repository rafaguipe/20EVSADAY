# Sons do EVSADAY

## Como adicionar os sons:

### 1. Som de Moeda (`coin.mp3`)
- **Quando toca**: Ao registrar um EV
- **Sugestões de busca**: "coin sound effect", "moeda som", "reward sound"

### 2. Som de Vitória (`victory.mp3`)
- **Quando toca**: Ao conquistar um badge
- **Sugestões de busca**: "victory sound", "achievement sound", "success sound", "fanfare"

## Como adicionar:

1. **Baixe os sons** (formato MP3, WAV ou OGG)
2. **Renomeie para `coin.mp3` e `victory.mp3`**
3. **Cole nesta pasta**

## Sugestões de sites para baixar sons:
- https://freesound.org/ (gratuito)
- https://mixkit.co/free-sound-effects/ (gratuito)
- https://www.zapsplat.com/ (gratuito com registro)

## Formato recomendado:
- **Formato**: MP3
- **Qualidade**: 128kbps ou superior
- **Duração**: 1-3 segundos
- **Volume**: Moderado (será ajustado no código)
