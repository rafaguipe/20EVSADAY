# Sons do EVSADAY

## Como adicionar o som de moeda:

1. **Baixe um som de moeda** (formato MP3, WAV ou OGG)
2. **Renomeie para `coin.mp3`**
3. **Cole nesta pasta**

## Sugestões de sites para baixar sons:
- https://freesound.org/ (gratuito)
- https://mixkit.co/free-sound-effects/ (gratuito)
- https://www.zapsplat.com/ (gratuito com registro)

## Formato recomendado:
- **Formato**: MP3
- **Qualidade**: 128kbps ou superior
- **Duração**: 1-3 segundos
- **Volume**: Moderado (será ajustado no código)

## Exemplo de busca:
- "coin sound effect"
- "moeda som"
